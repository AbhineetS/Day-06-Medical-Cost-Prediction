# Day 06 — Medical Cost Prediction (Regression)

Predict individual medical insurance charges using Linear Regression and Random Forest, with proper preprocessing (one-hot encoding + scaling) and evaluation (MAE, RMSE, R²).

**Run:**
1) `source ../Day-01-Titanic/venv/bin/activate`
2) `pip install pandas scikit-learn matplotlib seaborn`
3) `python3 run_medical_cost_model.py`

**Outputs:** printed metrics and `model_comparison.png`.
